# Post Templates For Faceless Film-Sound Content

Research date: April 19, 2026

## Purpose

This document converts the style research in `Faceless/docs/` into usable post templates for a faceless account about sound and music in TV and film.

Use this as a working template bank for:

- Instagram Reels
- TikTok
- YouTube Shorts
- caption drafting
- hook generation
- batch planning

Related docs:

- [AncientLegends style tutorial](./ancientlegends_style.md)
- [LesChanteryStudio style tutorial](./leschanterystudio_style.md)
- [Faceless film-sound house style](./faceless_film_sound_house_style.md)

## 1. How To Use These Templates

For each post:

1. Choose one scene.
2. Choose one sound mechanism.
3. Choose one template.
4. Rewrite the hook so it sounds like a verdict.
5. Keep the explanation focused on one deeper point only.

Each template below includes:

- best use case
- on-screen hook
- caption/body structure
- optional closing line

## 2. Fast Fill-In Fields

Use these fields before writing:

## 3. Template 01: One Sound Controls The Scene

Best for:

- endings
- reveals
- tension scenes
- scenes with one repeated cue

### On-Screen Hook

`This scene is controlled by one sound.`

### Caption

```text
In [film/show], this moment isn’t working because of [obvious thing].

It’s working because [specific sound cue] keeps shaping how we read the scene.

That one choice turns the moment from [state A] into [state B].
```

### Closing Line

`That’s why the ending lands like an impact, not just an image.`

## 27. Final Rule

Do not post until you can finish this sentence:

`This scene works because ____________________.`

If the answer is vague, the post is not ready.
